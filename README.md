### 技术栈

- **_Vue_** 全家桶 通过 Vue-CLI3 初始化生成。
- **_ElementUI_** 
- **_better-scroll_** 歌词滚动部分用了黄轶老大的
- **_CSS Variables_** 主题换肤。
- **_ESNext_** （JavaScript 语言的下一代标准）
- **_Sass_**（CSS 预处理器）
- **_postcss-pxtorem_**（自动处理 rem，自适应屏幕）
- **_workbox-webpack-plugin_** 谷歌开发的利用 Service Worker 实现页面预缓存的插件。
